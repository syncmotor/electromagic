# electromagic
Simulate rotating magnetic fields at synchronous speed.

Synchronous drives & inverters private limited repository.
http://www.syncmotor.com/

This repository includes computer programs to simulate rotating OR travelling magnetic fields produced from addition of three pulsating waveforms.

Run the files in sequence and experience the electromagic. 

* sync_one: illustrates the pulsating magnetic field produced when a 1-phase sinusoidally distributed winding in space, is supplied by 1-phase sinusoidally time-varying currents.
* sync_two: illustrates the pulsating magnetic field produced when a 3-phase sinusoidally distributed winding in space, is supplied by 3-phase sinusoidally time-varying currents.
* sync_three: illustrates the rotating magnetic field at synchronous speeds produced as a result of the above physics, discovered by Nikola Tesla.

Intended for educational purposes only. 

Current solutions:
1) MATLAB (all versions)
2) Python (in progress)
